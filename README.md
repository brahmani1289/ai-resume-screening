# AI-Powered Resume Screening System

An interactive Streamlit web app that automates resume screening using NLP and machine learning.  
It compares resumes against a given job description and ranks candidates based on similarity scores.

---

## 🚀 Features
- Upload resumes in PDF format
- Paste job description text
- Preprocess text with **NLTK stopwords** and **SpaCy**
- Compute similarity using **TF-IDF + Cosine Similarity**
- Candidate ranking dashboard with Streamlit UI

---

## 🛠️ Tech Stack
- **Python 3.14**
- **Streamlit** (UI)
- **SpaCy** (NLP preprocessing)
- **NLTK** (stopwords)
- **PyPDF2** (PDF parsing)
- **Scikit-learn** (TF-IDF, cosine similarity)
